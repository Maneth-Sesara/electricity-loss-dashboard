# Global Electricity Losses and Energy Efficiency Dashboard

This project is a Streamlit dashboard developed using World Bank data on electric power transmission and distribution losses (% of output).

## Features
- Country selection
- Year range filter
- Electricity loss trend chart
- Top 10 highest electricity loss countries
- Top 10 lowest electricity loss countries
- Filtered dataset table

## Dataset
World Bank: Electric power transmission and distribution losses (% of output)

## Tools Used
- Python
- Streamlit
- Pandas
- Plotly
